# cms-demo-php

#### 介绍
简单的cms小demo，php实现

### 目录
```
- config 配置文件
  - database.php 数据库配置

- model 数据模型
  - connect.php 连接数据库

- middleware 中间件
  - loginGuard.php 登录守卫

- route 路由
  - admin 后台路由模块
  - home  前台路由模块
  - admin.php 后台路由入口
  - home.php 前台台路由入口

- data 数据存储
  - session 会话存储
  - test.sql 测试用的sql文件

- public 静态资源目录
  - admin 后台静态资源
  - home 前台静态资源
  - uploads 文件上传目录

- views 视图模板
  - admin 后台模板
    - common 公共部分
  - home 前台模板
    - common 公共部分

- index.php 入口
```
